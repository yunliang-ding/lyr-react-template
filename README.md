## 安装后端依赖

```
npm install
```

## 启动服务

```
npm start

默认端口 7001
```

## 服务部署

```
npm run build

默认端口 7000

需要将pm2.json 的 cwd 只想项目根目录地址
```



## 前端服务

```
位置在 www 目录下采用 vite + react-ts
```
